eit-remote(1)
=============
:man source:   eit {eitversion}
:man manual:   eit {eitversion}


NAME
----
eit-remote - manage repositories


SYNOPSIS
--------
eit remote [-h] {list,add} ...


INTRODUCTION
------------
Manage (add, remove, list) configured repositories.



OPTIONS
-------
"eit remote" supports the following options which alters its behaviour.


OPTIONAL ARGUMENTS
~~~~~~~~~~~~~~~~~~
*--help*::
    show this help message and exit

ACTION
~~~~~~
*list*::
    list repositories

*add*::
    add repository



AUTHORS
-------
Eit has been accidentally prototyped during a rainy Sunday by Fabio
Erculiani who was looking for innovative ways of handling package
bumps in Argent Entropy repositories.
A few months later, Eit saw the light.
