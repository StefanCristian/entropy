eit-repo(1)
===========
:man source:   eit {eitversion}
:man manual:   eit {eitversion}


NAME
----
eit-repo - manage repositories


SYNOPSIS
--------
eit repo [-h] {show,add,remove} ...


INTRODUCTION
------------
Manage Entropy Server Repositories.



OPTIONS
-------
"eit repo" supports the following options which alters its behaviour.


OPTIONAL ARGUMENTS
~~~~~~~~~~~~~~~~~~
*--help*::
    show this help message and exit

ACTION
~~~~~~
*show*::
    show repositories and mirrors status

*add*::
    add a repository

*remove*::
    remove a repository



AUTHORS
-------
Eit has been accidentally prototyped during a rainy Sunday by Fabio
Erculiani who was looking for innovative ways of handling package
bumps in Argent Entropy repositories.
A few months later, Eit saw the light.
