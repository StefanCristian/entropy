equo-mask(1)
============
:man source:   equo {equoversion}
:man manual:   equo {equoversion}


NAME
----
equo-mask - mask one or more packages


SYNOPSIS
--------
equo mask [-h] [--ask | --pretend] <package> [<package> ...]


INTRODUCTION
------------
Mask packages so that installation and update will be inhibited.



OPTIONS
-------
"equo mask" supports the following options which alters its behaviour.


POSITIONAL ARGUMENTS
~~~~~~~~~~~~~~~~~~~~
*<package>*::
    package name

OPTIONAL ARGUMENTS
~~~~~~~~~~~~~~~~~~
*--help*::
    show this help message and exit

*-a*::
    ask before making any changes

*-p*::
    only show what would be done
