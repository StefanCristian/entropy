:man source:   equo {equoversion}

NAME
----
equo-update - update repositories


SYNOPSIS
--------
equo update [-h] [--force] [<repo> [<repo> ...]]


INTRODUCTION
------------
Update Entropy Repositories.



OPTIONS
-------
"equo update" supports the following options which alters its behaviour.


POSITIONAL ARGUMENTS
~~~~~~~~~~~~~~~~~~~~
*<repo>*::
    repository

OPTIONAL ARGUMENTS
~~~~~~~~~~~~~~~~~~
*--help*::
    show this help message and exit

*--force*::
    force update
